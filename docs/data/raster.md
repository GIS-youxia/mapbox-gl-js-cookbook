# 栅格
栅格数据一般发布为 `WMS`、`WMTS` 服务，使用 **`raster`** 格式加载；如果是一些图片格式，并且文件大小比较小，比如：温度图、湿度图、雷达图等，则可以根据经纬度范围使用 **`image`** 格式加载。

## raster
``` js
'source-id': {
  'type': 'raster',
  'tiles': [
    'https://stamen-tiles.a.ssl.fastly.net/watercolor/{z}/{x}/{y}.jpg'
  ],
  'tileSize': 256,
}
```
